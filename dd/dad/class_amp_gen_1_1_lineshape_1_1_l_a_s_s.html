Detailed Description

The LASS parameterisation of the \( K\pi \) S-wave is derived from fits to ~ elastic \( K \pi \) scattering data, which is approximately up to the \( K \eta^\prime \) threshold. In this regime, unitarity implies that phases, rather than amplitudes should be summed. In this context, a slow varying nonresonant phase,

\[ \tan(\phi_{NR}) = \frac{2 a q}{2 + arq^2}, \]

where \(a, r\) are parameters determined from scattering data, is added to the phase shift of a Breit-Wigner,

\[ \tan(\phi_{BW}) = \frac{m\Gamma(s)}{m^2 -s }, \]

normally associated to the \( K^*(1430)^{0} \) resonance. The total amplitude is therefore:

\[ \mathcal{A}(s) = \frac{2 a \sqrt{s} }{ 2 + arq^2 - 2iaq} + \frac{2+arq^2 + 2iaq}{2+arq^2 - 2iaq }\mathcal{A}_{BW}(s) = \mathcal{A}_{NR}(s) + \mathcal{A}^{\prime}_{BW}(s). \]

As this expression somewhat resembles the sum of a Breit-Wigner with a slowly varying nonresonant component, the two parts are sometimes split apart with an additional production amplitude placed on one or the other. These can be accessed separately using the modifiers LASS.BW and LASS.NR, and given independent coupling constants. From the user interface:

K*(1430)bar0[LASS.NR]{K-,pi+} 0 2.0 0.1 0 -90 0.1
K*(1430)bar0[LASS.BW]{K-,pi+} 0 1.5 0.1 0 20 0.1

Corresponds to the overall lineshape:

\[ \mathcal{A}(s) = \left(\texttt{K*(1430)bar0[LASS.NR]{K-,pi+}}\right) \mathcal{A}_{NR}(s) + \left(\texttt{K*(1430)bar0[LASS.BW]{K-,pi+}}\right) \mathcal{A}^{\prime}_{BW}(s) = 2.0 e^{-90^\mathrm{o}} \mathcal{A}_{NR}(s) + 1.5 e^{-20^\mathrm{o}} \mathcal{A}^{\prime}_{BW}(s). \]

An alternative way to introduce a different coupling between the two components is to fix one coupling and to introduce an additional production coupling. In the user interface, this might for example in \( D^{0} \to K^{-} \pi^{+} \pi^{0} \) decays:

D0{K*(1430)bar0,pi0} 0 1.3 0.0 0 15 0.1
K*(1430)bar0[LASS.NR]{K-,pi+} 0 2.0 0.1 0 -90 0.1
K*(1430)bar0[LASS.BW]{K-,pi+} 2 1.0 0.0 0 0 0.0

Then the lineshape with production couplings will be:

\[ \mathcal{A}(s) = \left(\texttt{D0{K*(1430)bar0,pi0}}\right)\left( = \left(\texttt{K*(1430)bar0[LASS.NR]{K-,pi+}}\right) \mathcal{A}_{NR}(s) + \mathcal{A}^{\prime}_{BW}(s) \right) = 1.3 e^{15^\mathrm{o}} \left( 2.0 e^{-90^\mathrm{o}} \mathcal{A}_{NR}(s) + \mathcal{A}^{\prime}_{BW}(s) \right). \]
